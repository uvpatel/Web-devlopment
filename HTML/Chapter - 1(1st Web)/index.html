Creating Your First Website

Introduction

We start building a website by creating a file named index.html. This is a special filename that is automatically loaded when the website root address is accessed.

A Basic HTML Page

<!DOCTYPE html>  <!-- Specifies this is an HTML5 document -->
<html>  <!-- Root of an HTML page -->
    <head>  <!-- Contains page metadata -->
        <title>Urvil's Website</title>  <!-- Title of the page -->
    </head>
    <body>  <!-- The main body of the page (rendered by the browser) -->
        <h1>This is h1</h1>
        <p>My paragraph</p>  <!-- Paragraph tag -->
    </body>
</html>

Important Notes

  • <head> and <body> tags are children of the <html> tag.
  • HTML is the parent of the <head> and <body> tags.
  • Most HTML elements have opening and closing tags with content in between.
  • Some HTML tags do not have content; these are called empty elements (self-closing tags), e.g., <br>.
  • You can use either .htm or .html as the file extension.
  • You can use "Inspect Element" or "View Page Source" in Chrome to check a website's HTML code.

Comments in HTML

Comments in HTML help document the source code and are ignored by the browser. 

<!-- This is an HTML comment -->

Case Sensitivity

HTML is case-insensitive, meaning <H1> and <h1> are treated the same.
