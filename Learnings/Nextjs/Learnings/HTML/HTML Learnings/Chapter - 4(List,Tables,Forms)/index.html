CHAPTER 4 – LISTS, TABLES & FORMS

HTML Lists

Lists are used to display information in a structured format. There are three types of lists in HTML:

We use the following tags to create lists:

            <ul> - Unordered List
            <ol> - Ordered List
            <dl> - Definition List
            <li> - List Item
            <dt> - Definition Term
            <dd> - Definition Description

Unordered List

An unordered list is a collection of items that appear in bullet points.

  • Item 1
  • Item 2
  • Item 3

Ordered List

An ordered list is a collection of items that appear in a numbered sequence.

  1. Item 1
  2. Item 2
  3. Item 3

Definition List

A definition list is used for terms and their descriptions.

HTML Tables

We use the <table> tag to create tables. A table consists of rows and columns, and we use the following tags to structure it:

            <thead> - Table Head
            <tbody> - Table Body
            <tr> - Table Row
            <th> - Table Header
            <td> - Table Data

HTML Forms

Forms in HTML are used to collect user input. The <form> tag is used to create a form, and it contains different types of input elements.

Common form elements:

            <input> - Input field
            <label> - Label for input field
            <button> - Button
            <select> - Dropdown list
            <textarea> - Multiline text input
