CHAPTER 2 – BASIC HTML TAGS

Below are examples of basic HTML tags with their implementation and output.

HTML ELEMENT

Definition: An HTML element consists of everything from the opening tag to the closing tag.

Implementation:

<p>This is a paragraph.</p>

Output:

This is a paragraph.

HTML ATTRIBUTES

Definition: HTML attributes provide additional information about an element.

Implementation:

<a href="https://example.com">Visit Example</a>

Output:

Visit Example

IMAGE TAG

Definition: The <img> tag is used to add images to a webpage.

Implementation:

<img src="example.jpg" alt="Example Image">

Output:

TEXT FORMATTING

Definition: These tags are used for formatting text.

Implementation:

<b>Bold Text</b> 
<i>Italic Text</i> 
<u>Underlined Text</u>

Output:

Bold Text
Italic Text
Underlined Text

LINE BREAK (BR TAG)

Definition: The <br> tag adds a line break in text.

Implementation:

<p>Hello<br>World</p>

Output:

Hello
World

BIG & SMALL TEXT

Definition: The <big> and <small> tags adjust text size.

Implementation:

<big>Big Text</big>
<small>Small Text</small>

Output:

Big Text
Small Text

HORIZONTAL LINE (HR TAG)

Definition: The <hr> tag creates a horizontal line to separate content.

Implementation:

<hr>

Output:

SUBSCRIPT & SUPERSCRIPT

Definition: The <sub> and <sup> tags adjust text position.

Implementation:

H<sub>2</sub>O (Subscript)
5<sup>2</sup> = 25 (Superscript)

Output:

H2O

52 = 25

PRE TAG (PRESERVE FORMATTING)

Definition: The <pre> tag preserves spaces and new lines in text.

Implementation:

<pre>
    This is written
    using pre
    tag
</pre>

Output:

    This is written
    using pre
    tag
